 Só os Fortes Suplementos 💪

Objetivo

O projeto Só os Fortes Suplementos tem como objetivo criar uma página web para uma loja fictícia de suplementos esportivos, oferecendo uma experiência simples e intuitiva para apresentação de produtos, informações sobre a empresa e contato com os clientes.
A aplicação foi desenvolvida para divulgar suplementos como Whey Protein, Creatina e Pré-Treino, permitindo que os usuários conheçam os produtos disponíveis, seus valores e detalhes, além de acessar informações institucionais da empresa.


Tecnologias utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

HTML

  * Criação da estrutura das páginas;
  * Organização dos conteúdos;
  * Criação de formulários e seções.

  CSS

  * Estilização da interface;
  * Criação do layout responsivo;
  * Personalização de cards, botões, páginas e elementos visuais.

  JavaScript

  * Interações da página;
  * Controle dos detalhes dos produtos;
  * Validação e funcionamento do formulário de contato.

O projeto possui as seguintes páginas:

 🛒 Produtos (`produtos.html`)
